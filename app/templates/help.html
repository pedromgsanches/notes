Markdown Help

Markdown is a simple formatting language that allows you to create formatted text easily. Here are the main elements you can use:

Headers

# Main Header
## Subheader
### Smaller Header
#### Very Small Header

Emphasis

*italic text* or _italic text_
**bold text** or __bold text__
~~strikethrough text~~

Lists

Unordered list:
* Item 1
* Item 2
  * Subitem 2.1
  * Subitem 2.2

Ordered list:
1. First item
2. Second item
3. Third item

Links and Images

[link text](URL)
![image alt text](image URL)

Quotes

> This is a quote.
>> It can have multiple lines.

Code

`inline code`

```python
def hello_world():
    print("Hello, world!")
```

Tables

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

Horizontal Rule

---
or
***
or
___

Tip: You can use the "Preview" button to see how your note will look before saving.
